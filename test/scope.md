# Scope
Scope is a concept that tells us where a variable may be accessed.

We have two major types of scope;
1.  Local scope
2.  Global scope

## Local scope
The scope of a variable is said to be local when it is defined within a block. Hence, cannot be accessed outside of that block, except for its descendants.

### Global scope
A variable is said to be in global scope, when it can be accessed by every other scope in a module.

> A Module is in its simplest form, same a JavaScript file.
